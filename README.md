## Common Technologies

All LEXA Family variants share core technological capabilities:

### AI Navigation System

| Algorithm | Function |
|-----------|----------|
| **Visual-Inertial Odometry (VIO)** | Camera+IMU fusion for dead reckoning |
| **Visual SLAM** | Real-time mapping and localization |
| **Terrain Matching** | Correlation with pre-loaded satellite imagery |
| **AI Target Recognition** | CNN-based object detection (YOLOv8/v9) |

### GPS-Denied Operation

All variants are capable of fully autonomous operation in GPS-denied environments using:
- Visual navigation fusion
- Terrain contour matching
- Inertial navigation dead reckoning
- Stellar navigation (M7-M9)

### Autonomous Behavior Modes

| Mode | Trigger | Behavior |
|------|---------|----------|
| **Normal** | GPS healthy | GPS-INS-Visual fusion |
| **GPS-Denied** | GPS loss detected | Visual SLAM + terrain matching |
| **Terminal** | Target acquired | AI track-and-intercept |
| **Emergency RTL** | Critical failure | Autonomous return to launch |

### Software Stack

| Component | Technology |
|-----------|------------|
| **Autopilot** | PX4 v1.14+ |
| **AI Framework** | NVIDIA JetPack |
| **Vision** | OpenCV 4.8+ |
| **AI Models** | YOLOv8/YOLOv9 |
